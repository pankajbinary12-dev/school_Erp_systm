# School ERP System - हिंदी गाइड

## 🎓 विशेषताएं

### ✅ पूर्ण कार्यक्षमता
- **मल्टी-यूजर लॉगिन सिस्टम** (छात्र और शिक्षक)
- **छात्र प्रबंधन** (प्रवेश, अपडेट, डिलीट, प्रमोट)
- **शिक्षक/स्टाफ प्रबंधन** (जोड़ें, अपडेट, डिलीट)
- **मास्टर डेटा प्रबंधन**
  - सत्र (शैक्षणिक वर्ष)
  - कक्षाएं
  - सेक्शन
  - विषय
- **छात्र डैशबोर्ड** व्यक्तिगत जानकारी के साथ
- **शिक्षक डैशबोर्ड** शेड्यूल और नोटिफिकेशन के साथ
- **प्रोफेशनल UI** Bootstrap 5 के साथ
- **AJAX-आधारित** ऑपरेशन (पेज रीलोड नहीं)
- **SweetAlert2** सुंदर नोटिफिकेशन के लिए
- **PostgreSQL डेटाबेस** सपोर्ट

## 🚀 इंस्टॉलेशन स्टेप्स

### आवश्यकताएं
- PHP 8.2 या उच्चतर
- Composer
- PostgreSQL 12 या उच्चतर
- Node.js & NPM (वैकल्पिक)

### स्टेप 1: फोल्डर में जाएं
```bash
cd school-erp-system
```

### स्टेप 2: सेटअप स्क्रिप्ट चलाएं
```bash
SETUP.bat
```

यह स्क्रिप्ट स्वचालित रूप से करेगी:
- Composer dependencies इंस्टॉल
- Application key जनरेट
- Database migrations चलाना
- Sample data सीड करना
- Storage link बनाना
- Cache क्लियर करना

### स्टेप 3: डेटाबेस कॉन्फ़िगरेशन

`.env` फ़ाइल में अपना PostgreSQL कॉन्फ़िगरेशन चेक करें:

```env
DB_CONNECTION=pgsql
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=school_erp
DB_USERNAME=postgres
DB_PASSWORD=pankaj123@
```

**नोट**: पहले PostgreSQL में `school_erp` नाम का डेटाबेस बनाएं:

```sql
CREATE DATABASE school_erp;
```

### स्टेप 4: सर्वर स्टार्ट करें
```bash
php artisan serve
```

ब्राउज़र में खोलें: `http://localhost:8000`

## 📊 डेटाबेस स्कीमा

### छात्र टेबल
- व्यक्तिगत जानकारी (नाम, जन्मतिथि, लिंग, आदि)
- शैक्षणिक जानकारी (कक्षा, सेक्शन, सत्र)
- अभिभावक जानकारी
- लॉगिन क्रेडेंशियल्स (यूजरनेम, पासवर्ड)
- फोटो अपलोड सपोर्ट

### शिक्षक टेबल
- व्यक्तिगत जानकारी
- व्यावसायिक जानकारी (योग्यता, ज्वाइनिंग डेट)
- लॉगिन क्रेडेंशियल्स
- फोटो अपलोड सपोर्ट

### मास्टर टेबल्स
- **सत्र**: शैक्षणिक वर्ष सक्रिय स्थिति के साथ
- **कक्षाएं**: कक्षा नाम और संख्यात्मक मान
- **सेक्शन**: प्रति कक्षा सेक्शन क्षमता के साथ
- **विषय**: विषय विवरण कोड के साथ

## 🎓 उपयोग गाइड

### छात्र प्रवेश कैसे करें:
1. Admin पैनल में जाएं

3. "Add New Student" बटन क्लिक करें
4. सभी आवश्यक जानकारी भरें

5. "Save" बटन क्लिक करें

1. "Students" पेज पर जाएं
